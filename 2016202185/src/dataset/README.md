## 中文语音识别
##### 1. 环境
- Python：3.5
- Tensorflow : 1.5.0

------

##### 2. 训练数据下载
- 清华大学中文语料库（thchs30）[下载地址](http://www.openslr.org/18/)

------

##### 3. 训练
- 配置 conf 目录下的 conf.ini 文件中的各项
- 在终端运行 ```python train.py``` 开始训练
- 在终端运行 ```python test.py``` 测试
- 也可以使用PyCharm打开
